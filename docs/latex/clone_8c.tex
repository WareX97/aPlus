Creates a new process.~\newline
Unlike fork(2), these calls allow the child process to share parts of its execution context with the calling process, such as the memory space, the table of file descriptors, and the table of signal handlers.~\newline
The main use of \hyperlink{clone_8c_a14a069ba6b991078484ac2b7573a5f23}{sys\+\_\+clone()} is to implement threads\+: multiple threads of control in a program that run concurrently in a shared memory space.
