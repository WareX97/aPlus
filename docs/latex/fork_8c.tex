fork() creates a new process by duplicating the calling process.~\newline
 The new process, referred to as the child, is an exact duplicate of the calling process, referred to as the parent. 

\begin{DoxyReturn}{Returns}
On success, the P\+I\+D of the child process is returned in the parent, and 0 is returned in the child.~\newline
 On failure, -\/1 is returned in the parent, no child process is created, and errno is set appropriately. 
\end{DoxyReturn}
